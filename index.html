навыки

HTML

Изучил теги и основные атрибуты. Применяю семантические теги. Проверю разметку на валидность через w3.org.

CSS / Flexbox / CSS Modules

Научился применять Flexbox для удобного создания гибких контейнеров и макетов. Освоил CSS Modules для разделения CSS на независимые файлы при работе с React. Изучил использование медиа запросов для адаптивной верстки. Научился создавать анимации и использовать ключевые кадры и переходы.

Sass (Scss)

Изучил подход к организации SCSS файлов с помощью шаблона 7-1. Так же освоил и использую на практике такие возможности как вложенность селекторов, переменные, миксины, математические операторы, обработка ошибок, условия.

JavaScript

Помимо базы изучил работу с объектами, прототипы, особенности this, привязку контекста. Понимаю как работает рекурсия, стек. При необходимости использую замыкания. Умею работать с асинхронностями. Понимаю принцип работы Event Loop. Слежу за обновлениями языка, знаю новшества ES11.

ReactJS

Освоил базовые составляющие React: компоненты, пропсы, стейт, однонаправленный поток, жизненный цикл, подъем состояния. Использую функциональные компоненты + хуки. Но так же изучал и классовые компоненты. Изучил и активно использовал контекст в одном из проектов. Понимаю как работает согласование.

Git / GitHub

Научился работать система контроля версий Git. Умею работать с ветками, репозиториями. Так же пушить, пулить, коммитить, использовать файл gitignore.

Redux

Понимаю принципы и назначение Redux. Сейчас изучаю работу со стором, экшенами, редьюсерами и их взаимодействием.
